-GATE CSE Question Classification Dataset

--Overview

This dataset contains 2,400 past GATE Computer Science (CSE) examination questions categorized into 8 key topics. 300 questions in each topic. It is designed for Natural Language Processing (NLP) tasks such as question classification, text analysis, and topic modeling.
The dataset can be used to train machine learning models to classify unseen GATE questions into their respective topics, perform exploratory data analysis (EDA), or develop automated learning tools.

--Dataset Details

   -Total Questions: 2,400
   -Number of Categories: 8
   -File Format: CSV (Comma-Separated Values)

--Categories Included
  1. Computer Networks
  2. Operating Systems
  3. Mathematics
  4. General Aptitude
  5. Programming & Data Structures
  6. Computer Organization & Architecture
  7. Digital Logic
  8. Theory of Computation

--Column Descriptions
-------------------------------------------------------------------------
Column      |   Description |    Example
--------------------------------------------------------------------------
Topic       |The subject category of the question | "Computer Networks"
--------------------------------------------------------------------------
Question    |The actual question text, without options | "What is the purpose of the OSI model in networking?"
------------------------------------------------------------------------------------------

--Example Usage

Here are some ways you can use this dataset:

  -Training a Machine Learning Classifier
  -Use the dataset to train an NLP model to automatically classify GATE questions.

  -Educational Tools & Exam Preparation: Develop personalized mock test generators or GATE exam preparation apps.
